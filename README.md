# 消費者物価指数（CPI）分析アプリ

## 概要

本アプリは、政府統計ポータルサイト **e-Stat** が公開している  
**消費者物価指数（CPI：2020年基準）** のデータを用い、  
地域別・品目別の物価動向を可視化するための Streamlit アプリです。

単年における「指数・前年同月比」の比較と、  
複数年にわたる「時間推移」の可視化を切り替えて表示できるよう設計しています。  
これにより、物価水準の変化を **静的比較と時系列分析の両面**から確認できます。

---

## データについて

### 1. 統計表名  
2020年基準消費者物価指数

### 2. 表番号・データセット名  
表番号 1  
消費者物価指数（2020年基準）

### 3. データ取得日  
2026年1月26日（月）

### 4. 各列の意味

| 列名 | 内容 |
|---|---|
| 地域（2020年基準） | 集計対象の地域名（全国、地方、東京都区部など） |
| 時間軸（年・月） | 観測時点（例：2024年6月、2025年12月） |
| 2020年基準品目 | 集計対象の品目（総合、食料、光熱・水道など） |
| 指数 | 2020年を100とした消費者物価指数 |
| 前年同月比【%】 | 前年同月と比較した物価の増減率（％） |

### 5. 単位・注意点

- **指数**  
  - 単位：なし（指数値）  
  - 2020年を100とした相対的な物価水準を表す  
- **前年同月比【%】**  
  - 単位：％  
  - プラスは上昇、マイナスは下落を示す  

**注意点**
- 本データは **6月・12月のみの半年データ**であり、月次データではありません。
- そのため、短期的な月変動の分析には適していません。

## 未使用UI部品の利用について

本アプリでは、グラフ表示以外にも以下の Streamlit UI 部品を使用しています。

### st.set_page_config  
- **使用目的**：ページタイトル・アイコン・レイアウトを設定し、アプリ全体の視認性を向上させるため  
- **使用箇所**：アプリ起動時の基本設定

### st.caption  
- **使用目的**：データの出典や補足情報を簡潔に提示するため  
- **使用箇所**：アプリタイトル直下の説明文

### st.divider  
- **使用目的**：サイドバー内の設定項目を視覚的に区切り、操作性を高めるため  
- **使用箇所**：表示内容選択と抽出条件の区切り

### st.expander  
- **使用目的**：CPI に関する補足説明を、必要な場合のみ表示できるようにするため  
- **使用箇所**：「消費者物価指数（CPI）とは」の説明部分

### st.warning  
- **使用目的**：指定条件に該当するデータが存在しない場合に、ユーザーへ明確に通知するため  
- **使用箇所**：データ抽出結果が空の場合
